# Vue Recipes Website
```
This is a Vue.js application that displays recipes and allows users to browse, search, and save their favorite recipes. 
The application consumes a REST API from a Laravel backend.
```

## Getting Started
```
To get started with this application, follow these steps:

1.Clone the repository to your local machine using the command git clone https://github.com/your-username/vue-recipes.git.
2.Install the dependencies by running npm install in the project root directory.
3.Create a .env file in the project root directory and set the following environment variables:
4.VUE_APP_API_URL - the URL of the Laravel backend API (e.g. http://localhost:8000/api)
5.Start the development server by running npm run serve.
6.Open the application in your web browser by navigating to http://localhost:8080.
```

### Features
```
This application includes the following features:

1.Browse recipes by category (e.g. appetizers, main dishes, desserts)
2.Search for recipes by name or ingredients
3.View recipe details, including ingredients and instructions
4.Save recipes to a favorites list
```

### Technologies Used
```
This application was built using the following technologies:

.Vue.js - front-end JavaScript framework
.Vuex - state management library for Vue.js
.Vue Router - client-side routing library for Vue.js
.Axios - Promise-based HTTP client for the browser and Node.js
.Bootstrap - front-end component library
.Laravel - back-end PHP framework
.MySQL - relational database management system
```
